### RSettingsQT
Redis based Settings for QT
#### features
- notify when setting changed
- async
#### install / build
```shell script
./build.sh
```
#### dependencies
- [QT](https://www.qt.io)
- [disqt](https://github.com/smthnspcl/disqt)
    - [cpp_redis](https://github.com/cpp-redis/cpp_redis)
        - [tacopie](https://github.com/cylix/tacopie)
